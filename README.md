# AI Counsellor Frontend

A modern React-based frontend for the AI Study Abroad Counsellor application.

## Features
- **AI Chat Interface**: Interactive chat with Google Gemini-powered counsellor.
- **University Discovery**: Search and explore universities with AI insights.
- **Shortlisting**: Manage your dream, target, and safe schools.
- **Application Guidance**: Track application progress, documents, and timelines (AI-generated).
- **Profile Management**: Build your student profile for personalized recommendations.

## Tech Stack
- **Framework**: React + Vite
- **Styling**: Tailwind CSS + DaisyUI
- **State Management**: Context API
- **Routing**: React Router
- **Icons**: Heroicons

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository
   ```original
   git clone https://github.com/Golu-Guptha/AI-Consellor-frontend.git
   ```
2. Install dependencies
   ```original
   npm install
   ```
3. Create `.env` file
   ```original
   VITE_API_URL=http://localhost:3001
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_key
   ```
4. Run development server
   ```original
   npm run dev
   ```
